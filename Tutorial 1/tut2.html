What exactly is HTML?

HTML, also known as Hyper Text Markup Language is a language used to create a website. Well, the entire functional and beautiful website needs other languages to be added but HTML works as a skeleton or support for the website without which an website cannot be created.

What exactly is CSS?

CSS, also known as Cascading Style Sheets is a language which is simply used to add beauty to a HTML website. When HTML acts as a skeleton, it cannot add required beauty or style to the actual website. So, for proper beauty and style to make the website look more attractive, CSS is used.

What exactly is JS?

JS, also known as Java Script is a programming language which is used to add logic or simply a brain to a website. While adding HTML & CSS can make the website look beautiful and attractive, it cannot make the website more interactive or functional which is why, Java Script is used on websites.
